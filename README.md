# 🤖 Lizo Finance Bot

بوت تيليغرام لتسجيل فواتير ومصروفات متجر Lizo تلقائياً في Google Sheets.

## كيفية التشغيل على Railway

1. ارفع هذا المجلد على GitHub
2. افتح railway.app وأنشئ مشروع جديد من GitHub
3. اختر الـ repo وانشر

## ما يفعله البوت

- 📸 يقرأ صور الفواتير تلقائياً
- 🧠 يصنفها (فاتورة شراء / مبيعة / تحويل صادر / وارد)
- 📊 يسجلها في Google Sheets فوراً
- ✅ يرسل تأكيد في القروب
